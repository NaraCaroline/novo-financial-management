### [Atualizado] Desafio Beca - Construção de uma API de Gestão Financeira.

### Docker
docker-compose up -d

### Swagger
Acesse: http://localhost:8081/swagger-ui.html

### - Endpoints -

### User Service (8081)
- POST `/users` - Cadastrar usuário
- POST `/login` - Autenticar
- PUT `/users/me` - Atualizar perfil
- DELETE `/users/me` - Deletar conta
- POST `/users/import` - Importar Excel

### Tecnologias

- Java 17
- Spring Boot 3
- PostgreSQL
- Docker
- JWT
- Apache POI
